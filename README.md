# dockerfile

------------------
Setup Instructions
------------------

1. Installing docker

  * Ubuntu
  ```sh
  sudo apt-get install docker.io
  ```

  * Fedora
  ```sh
  sudo yum install docker-io
  ```

2. Adding the current user to docker group

```sh
sudo gpasswd -a `whoami` docker
```
